# LT Recepciones  
> Sistema web profesional de recepción y gestión de usuarios.

[![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=nextdotjs&logoColor=white)](https://nextjs.org/)

[![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)

Este es un proyecto **Full Stack** desarrollado de forma independiente, abarcando desde el diseño de la arquitectura hasta el despliegue final. El sistema permite gestionar de manera eficiente el flujo de clientes desde un panel administrativo y facilitar la comunicación directa vía WhatsApp para atención inmediata.

##  Funcionalidades Destacadas
* **Gestión de Catálogo Inteligente:** Implementación de lógica de ordenamiento alfabético desde la base de datos para garantizar una navegación fluida.
* **Sencilla y eficaz Experiencia de Usuario (UX):** Desarrollo de selectores de cantidad dinámicos con validación de inventario en tiempo real y alertas visuales.
* **Sistema de Recepción:** Formulario de registro automatizado con redirección inteligente a WhatsApp.
* **Panel Administrativo Proporcional:** Dashboard para la visualización de registros y analíticas de visitas diarias.
* **Arquitectura Responsive:** Interfaz adaptada minuciosamente para escritorio y dispositivos móviles, respetando la identidad visual corporativa.

##  Stack Tecnológico

### Frontend
* **React / Next.js:** Interfaz dinámica con renderizado optimizado.
* **Tailwind CSS:** Diseño moderno basado en componentes.
* **TypeScript:** Tipado estático estricto en el 98% del código para máxima estabilidad.

### Backend
* **NestJS:** Framework estructurado para una lógica de negocio escalable.
* **Prisma ORM:** Manejo avanzado de entidades y consultas eficientes a la base de datos.
* **PostgreSQL:** Base de datos relacional para la persistencia segura de la información.

### DevOps & Calidad
* **Docker & Docker Compose:** Contenedorización completa del entorno de desarrollo.
* **Git/GitHub:** Control de versiones bajo estándares de la industria.
